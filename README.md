# 3d_viewer
Программа на C++(Qt) для просмотра каркасных моделей.

Данная программа позволяет загружать файлы с расширением .obj и просматривать каркасные модели с различными модификациями отображения (при парсинге данные о цветах игнорируются, используются только координаты вершин). Для аффинных преобразований использовалась собственная библиотека для работы с матрицами, а для .

Краткие возможности:

* Аффинные преобразования над моделью:
  * перемещение модели вдоль осей (x, y, z);
  * поворот вокруг осей фигуры;
  * скалирование модели;
* Изменения отображения:
  * толщина линий;
  * тип линии (пунктир, сплошная);
  * цвет линии;
  * отображение точек (без точек, круглые, квадратные);
  * цвет точек;
  * толщина точек;
  * цвет фона;
* Ренедер отображения в формате bmp, jpeg;
* Запись gif (10 фпс, 5 секунд, 640 х 480);
* После закрытия программы сохраняется ее состояние перед закрытием (откроете повторно - загрузится тот же файл, а модель будет в том же состоянии);

## Компиляция программы

Используйте команду: 
* <code>qmake6 3d_viewer.pro</code>

Цели полученного Makefile:

* <code>all</code>: установка;
* <code>dvi</code>: генерация документации;
* <code>dist</code>: архивирование src;

Подробный функционал программы предоставлен в генерируемой документации с помощью цели <code>dvi</code>.
